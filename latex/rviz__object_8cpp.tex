Method for publishing a cylinder in rviz. 

This Method creates a marker message to create a cylinder in rviz. The frame where the marker in will be plotted in rviz is the base frame of the gripper. The marker will be deleted if there is no object in the range of the sensor. The type of the marker is a cylinder. Marker will be placed on the y-\/axis on the measured distance from the origin The size of the marker is determined with the diameter and height variable.
