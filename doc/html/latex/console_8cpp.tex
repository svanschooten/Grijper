The main method of the console. The console listens to input an passes the commands on to the controller. 

The console listens to user input to generate a command. These commands can be 'open', 'close', 'relax', 'force $<$target force$>$=\char`\"{}\char`\"{}$>$', 'get\-\_\-force' and 'exit' The 'open' command manually opens the gripper. The 'close' command manually closes the gripper. The 'relax' command relaxes the gripper (e.\-g. set the target force to 0). This can only be done manually. The 'force $<$target force$>$=\char`\"{}\char`\"{}$>$' command sets the target force to the '$<$target force$>$=\char`\"{}\char`\"{}$>$' parameter. The 'get\-\_\-force' command returns the target force as has been set. The 'exit' command passes on the shutdown command to all listening nodes before shutting down itself. This should be used to control all nodes.

If an invalid command is given it will return \char`\"{}\-Please use the commands 'open', 'close', 'relax', 'force', 'get\-\_\-force' or 'exit'.\char`\"{}.'
